# Flappy Bird Clone
A pet & self-training project using raylib 5.5 to replicate Flappy Bird Game

### Resources:
- [raylib 5.5](raylib.com)
- [Flappy Bird assets](https://github.com/samuelcust/flappy-bird-assets)

### Project Structure
- **assets**: Contains all necessary audio and image files for the game look 
- **include**: raylib header files
- **lib**: raylib library file (**libraylib.a**)
- **src**
    + **header**: header files for game components (Bird, Ground, Pipes, ScoreManager)
    + **main.cpp**: Main game loop and basic calculations for the game
